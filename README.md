\# CertiSendPro 🚀



\*\*CertiSendPro\*\* es una aplicación web moderna diseñada para la gestión, generación y automatización del envío de certificados digitales masivos. Construida con tecnologías de vanguardia y potenciada por Inteligencia Artificial para optimizar procesos administrativos de certificación.

\## 🛠️ Tecnologías Utilizadas



El proyecto fue desarrollado utilizando el siguiente ecosistema técnico:

\* \*\*Frontend:\*\* React con TypeScript, estructurado de forma eficiente con Vite.

\* \*\*Inteligencia Artificial:\*\* Google AI Studio para automatizaciones inteligentes y procesamiento de datos.

\* \*\*Backend \& Hosting:\*\* Firebase (Authentication, Hosting y servicios en la nube para resguardar la consistencia de los datos).



\## ✨ Características Principales



\* \*\*Autenticación Segura:\*\* Ingreso controlado de usuarios a través de Firebase Auth.

\* \*\*Interfaz Dinámica e Intuitiva:\*\* Diseño responsivo adaptado para cargas de plantillas y datos de forma ágil.

\* \*\*Procesamiento Inteligente:\*\* Integración de modelos de lenguaje para optimizar la estructura de envío o emparejamiento de listas.

\* \*\*Despliegue Continuo:\*\* Configurado y optimizado directamente sobre Firebase Hosting.



\## 🚀 Instalación y Configuración Local



Si deseas ejecutar este proyecto en tu entorno de desarrollo local, sigue estos pasos:



\### 1. Clonar el repositorio

```bash

git clone git@github.com:LeonardoIAConsult/CertiSendPro.git

cd CertiSendProV2

```



\### 2. Instalar dependencias

```bash

npm install

```



\### 3. Configurar variables de entorno

Crea un archivo `.env` en la raíz del proyecto basándote en el archivo `.env.example` e introduce tus credenciales de Firebase y Google AI Studio:

```env

VITE\\\\\\\\\\\\\\\_FIREBASE\\\\\\\\\\\\\\\_API\\\\\\\\\\\\\\\_KEY=tu\\\\\\\\\\\\\\\_api\\\\\\\\\\\\\\\_key

VITE\\\\\\\\\\\\\\\_GOOGLE\\\\\\\\\\\\\\\_AI\\\\\\\\\\\\\\\_API\\\\\\\\\\\\\\\_KEY=tu\\\\\\\\\\\\\\\_ai\\\\\\\\\\\\\\\_studio\\\\\\\\\\\\\\\_key

```



\### 4. Ejecutar el servidor de desarrollo

```bash

npm run dev

```
